1. SKILL EXTRACTION (STRICT RULES)

---

Extract skills from the resume using the following rules:

* Only extract **core professional skills**, not tasks or responsibilities.

* Avoid micro-level fragmentation (e.g., do NOT split "HRIS modules" into multiple skills).

* Merge similar skills into one standardized skill:

  * Example:

    * "Recruitment", "Hiring", "Talent Acquisition" → "Talent Acquisition & Recruitment"
    * "Payroll", "Compensation", "Salary processing" → "Payroll & Compensation Management"

* Convert skills into **industry-standard naming conventions**.

* Remove duplicates completely.

* Prefer **high-level grouped skills** instead of raw phrases.

* ❗ LIMIT: Extract **minimum 8 and maximum 18 high-quality grouped skills only**.

* Prioritize most relevant and impactful skills.

* DO NOT include:

  * Daily tasks (e.g., "maintaining records", "report preparation")
  * Repetitive variations of same skill
  * Overly granular system modules

* Tools can be included ONLY if important (e.g., Excel, SAP, HRIS).
